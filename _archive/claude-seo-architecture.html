3. Flux detaille — /seo audit (execution parallele)

%%{init: {'theme': 'dark'}}%%
sequenceDiagram
    participant U as Utilisateur
    participant R as Routeur SKILL.md
    participant F as fetch_page.py
    participant P as parse_html.py
    participant T as Agent Technical
    participant C as Agent Content
    participant S as Agent Schema
    participant SM as Agent Sitemap
    participant PF as Agent Performance
    participant V as Agent Visual
    participant G as Agent GEO
    participant L as Agent Local

    U->>R: /seo audit https://example.com
    R->>F: Fetch homepage
    F-->>R: HTML + headers + redirects
    R->>P: Parse HTML
    P-->>R: Structure SEO extraite
    R->>R: Detect business type

    par Execution parallele (7-10 agents)
        R->>T: Crawl + audit technique
        R->>C: Analyse E-E-A-T
        R->>S: Validation schema
        R->>SM: Analyse sitemap
        R->>PF: Core Web Vitals
        R->>V: Screenshots 4 viewports
        R->>G: AI crawler + citability
        R->>L: GBP + NAP (si local)
    end

    T-->>R: Resultats technique
    C-->>R: Resultats contenu
    S-->>R: Resultats schema
    SM-->>R: Resultats sitemap
    PF-->>R: Resultats CWV
    V-->>R: Screenshots
    G-->>R: Resultats GEO
    L-->>R: Resultats local

    R->>R: Score SEO 0-100 (pondere)
    R-->>U: FULL-AUDIT-REPORT.md + ACTION-PLAN.md
