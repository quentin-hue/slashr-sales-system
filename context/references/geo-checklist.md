# GEO (Generative Engine Optimization) — Checklist

> Reference on-demand. Chargee quand GEO est dans le perimetre du deal.

## Objectif
Optimiser la visibilite du prospect dans les reponses IA (Google AI Overviews, ChatGPT, Perplexity, Copilot).

## Checklist technique

### Accessibilite aux crawlers IA
- [ ] robots.txt : pas de blocage des bots IA (GPTBot, Google-Extended, CCBot, anthropic-ai)
- [ ] llms.txt present a la racine (description du site pour les LLMs)
- [ ] Sitemap XML a jour et accessible

### Contenu citable
- [ ] Definitions claires en debut d'article (format "snippet-ready")
- [ ] FAQ structurees (question + reponse concise)
- [ ] Listes a puces pour les processus/etapes
- [ ] Tableaux comparatifs (facilement extractibles par les LLMs)
- [ ] Donnees chiffrees sourcees (les LLMs preferent les faits verifiables)

### Donnees structurees
- [ ] Schema JSON-LD sur les pages cles (Organization, Product, Article, FAQ, HowTo)
- [ ] Informations coherentes entre le schema et le contenu visible

### Autorite et citabilite
- [ ] Nom de marque mentionne dans le contenu (pas seulement dans le header)
- [ ] Sources externes qui mentionnent la marque
- [ ] Contenu original (pas de copie/paraphrase d'autres sites)

## Metriques de suivi
- Presence dans les AI Overviews (monitoring DataForSEO)
- Mentions LLM (ChatGPT, Perplexity) via DataForSEO AI Optimization
- Citations dans les reponses IA avec lien vers le site
